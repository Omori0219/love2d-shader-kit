# Code of Conduct

This project is intended to be a useful and welcoming resource for people of
all experience levels.

## Expected behavior

- Discuss ideas and code without attacking people.
- Give feedback that is specific, actionable, and proportionate.
- Respect different backgrounds, tools, and levels of shader experience.
- Credit prior work and disclose licenses when adapting an external technique.

## Unacceptable behavior

Harassment, discriminatory language, threats, sustained disruption, and
publishing another person's private information are not accepted.

## Enforcement

Report conduct concerns privately to the repository maintainers. Maintainers
may edit or remove content, close discussions, or restrict participation when
needed to keep the project safe and productive.
